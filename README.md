Salve + Vidas Administrador
Esse projeto é um complemento ao outro projeto "SalveVidas" que foi o meu projeto de TCC que apresentei na 
faculdade, a ideia era criar um aplicativo que permiticia aos usuarios criarem campanhas de doação de sangue com 
base na sua localização cadastrada. Esse projeto serviria mais como uma facilitação pois ele permite o cadastro de 
hospitais e usuarios atraves do aplicativo ao invés de ser diretamente no banco de dados através de inserts. 
O código em si não foi muito bem estruturado igual ao outro projeto, porém funcionou perfeitamente para a ideia que tive.
